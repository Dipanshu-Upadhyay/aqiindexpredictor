# AQI Index Predictor 🌍

A machine learning web application that predicts Air Quality Index (AQI) using historical air quality data.

## Features
- Predicts AQI using a trained ML model
- User-friendly web interface
- Trained on historical AQI data
- Built with Python

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
## Project Structure

```
aqi_webapp/
│── app.py
│── train_model.py
│── requirements.txt
│── data/
│   └── city_day.csv
```

## Installation

```bash
pip install -r requirements.txt
python train_model.py
python app.py
```

## Dataset

The project uses the `city_day.csv` dataset.
